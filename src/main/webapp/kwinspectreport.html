In Klocwork v9.6, the kwinspectreport xml export feature was removed. If you are using a Klocwork installation of v9.6 or later, you must tick this box or you may see a build error. The Klocwork issues for v9.6 and later are now retrieved using the web API. Please see "Publish Klocwork test result report" under the "Post-build Actions" section.
