Here you can specify options to be used when running kwbuildproject. In the options box please enter the option (ex. --jobs-num) and in the second box please enter the value (ex. 2).
To create a unique Klocwork tables directory in another location using the "-o" or "--tables-directory" options, use environment variables such as "${BUILD_ID}" and "${BUILD_NUMBER}" which will be replaced at runtime by the current build values.
e.g. -o C:\kloTables\${BUILD_NUMBER}.
You can also specify options which do not require a value, e.g. --verbose.
Do not specify the following options:
  • --host or -h
  • --port or -p
  • --project or -S
  • --url
  • --license-host or -H
  • --license-port or -P
